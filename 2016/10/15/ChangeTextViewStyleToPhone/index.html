给UITextField 设置手机号码的样式显示

目标 : 让textField 在输入手机号时, 显示成这种格式: [136 6666 6666] –> 第三位号码和第七位后面加一个空格, 在输入11为数字之后不可以继续输入.

方法 : 为textField 添加一个监听事件, 在长度为 3 和 8 的时候, 在textField.text 后面添加一个@” “, 在textField.text.length 大于 13 的时候不再显示继续输入的数字.

代码 : 

- (void)phoneTextFieldDidChange:(UITextField *)textField {
    if (textField.text.length > 13) {
            textField.text = [textField.text substringToIndex:13];
     }
      if (textField.text.length == 3 || textField.text.length == 8) {
        textField.text = [textField.text stringByAppendingString:@" "];
      }
}

BUG : 但是如果只是这样写, 便会在删除数字的时候遇到BUG –> 那就是空格不能被删除, 因为在光标到达空格后面时每按一次回退键, 上述代码便会在继续加一个空格…..

方法 : 在textField 的代理方法里监听回退键, 在光标到达空格的时, 按下回退键便直接 “删除” 2 位, “136 “ –> “13”.

代码 : 

- (BOOL)textField:(UITextField *)textField shouldChangeCharactersInRange:        (NSRange)range replacementString:(NSString *)string {
    if (textField == _phoneTextField) {
        if ([string isEqualToString:@""] && (textField.text.length == 4 ||                 textField.text.length == 9)) {
            textField.text = [textField.text                 substringToIndex:textField.text.length - 1];
          }
      }
     return YES;
}
